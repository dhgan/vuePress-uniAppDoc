### 就像数 1, 2, 3 一样容易 

``` bash      

  # 安装
  npm install uni-simple-router #  或者： yarn add uni-simple-router

  #  开始开发
  npm run dev

  #  构建静态文件
  npm run build

```
::: warning 注意
请优先使用HBuilderX进行初始化项目，脚手架项目请手动配置vue-config.js
:::
