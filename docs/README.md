### 就像数 1, 2, 3 一样容易 

``` bash      

  # 安装
  npm install uni-simple-router #  或者： yarn add uni-simple-router

  #  开始开发
  npm run dev

  #  构建静态文件
  npm run build

```
::: warning 注意
vue-cli 项目请手动配置 vue.config.js 下 transpileDependencies:['uni-simple-router'] [详细](https://cli.vuejs.org/zh/config/#transpiledependencies)
:::
